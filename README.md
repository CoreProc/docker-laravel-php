# PHP-FPM 8.5 (Alpine)

Base PHP image for CoreProc Laravel apps: `coreproc/laravel-php:8.5-fpm-alpine` (linux/amd64 + linux/arm64).

Includes common PHP extensions, Composer, Supercronic, and supervisor. See the `main` branch README for the full variant list.

Build and push:

```bash
make build-push
```
